# Predicting EmiratiTel Churn using Machine Learning

## 📌 Overview
This project focuses on predicting customer churn for EmiratiTel using Machine Learning techniques. The goal is to identify customers who are likely to leave the telecom company and provide data-driven recommendations to improve customer retention.

## 🎯 Objectives
- Predict customer churn using ML models
- Identify the main factors causing churn
- Help telecom companies reduce customer loss
- Support business decision-making using predictive analytics

## 📊 Dataset
The dataset contains:
- 7043 customer records
- 21 features
- Numerical and categorical variables

Features include:
- Customer demographics
- Service subscriptions
- Billing information
- Contract type
- Payment methods
- Internet services

Target Variable:
- `Churn` → Yes / No

## 🛠️ Tools & Technologies
- RapidMiner
- Machine Learning
- Data Cleaning & Preprocessing
- Logistic Regression
- Random Forest

## 🔍 Key Insights
Some important findings:
- Customers with low tenure are more likely to churn
- High monthly charges increase churn probability
- Fiber optic users churn more frequently
- Customers using automatic payment methods are more stable

## 🤖 Models Used
### Logistic Regression
Used as a baseline classification model for churn prediction.

### Random Forest
Used to improve prediction performance and identify feature importance.

## 📈 Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- AUC Score
- Confusion Matrix

## 💡 Business Recommendations
- Focus retention campaigns on high-risk customers
- Encourage long-term contracts
- Improve pricing strategies
- Promote automatic payment systems
